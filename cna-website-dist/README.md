# وكالة الأنباء التنسيقية (CNA)
## Coordinating News Agency — Official Website

---

### متطلبات التشغيل
- Node.js v16 أو أحدث

### التشغيل
```bash
npm install
npm start
# http://localhost:3000
# admin / admin123
```

---

### الميزات الكاملة

| الميزة | التفاصيل |
|--------|----------|
| الأخبار العاجلة | شريط متحرك ثابت بمحرك CSS animation |
| البث المباشر LIVE | دعم YouTube وأي iframe — شارة LIVE نابضة |
| خبر مميز | لون / تدرج / صورة خلفية قابلة للتخصيص |
| ثنائية اللغة | كل خبر يدعم عربي + إنجليزي |
| محرر النصوص | Quill.js محلي بالكامل — يعمل بدون إنترنت |
| التصويت | للمسجلين فقط — نتائج حقيقية |
| تسجيل الزوار | حسابات visitor مع صلاحية التصويت |
| الأخبار العالمية | NewsAPI.org — اختيار الدولة والتصنيف |
| الشعار | مسار محلي `/public/img/logo.svg` قابل للاستبدال |
| الخطوط | Cairo + Amiri — محلية بالكامل في `/public/fonts/` |
| العداد | تصاعدي / تنازلي مع ألوان وصور |

### الأخبار العالمية (NewsAPI)
1. اذهب إلى [newsapi.org](https://newsapi.org) وأنشئ حساباً مجانياً
2. انسخ المفتاح (API Key)
3. في لوحة الإدارة → الإعدادات → الأخبار العالمية → الصق المفتاح
4. اختر الدولة والتصنيف الافتراضي
5. المستخدمون يمكنهم تغيير الدولة والتصنيف مباشرة من الصفحة الرئيسية

### استبدال الشعار
- ضع ملف الشعار في `public/img/logo.svg` (أو PNG)
- أو من لوحة الإدارة → الإعدادات → رفع شعار جديد

### أدوار المستخدمين
| الدور | الصلاحيات |
|-------|-----------|
| `admin` | كامل الصلاحيات |
| `editor` | إنشاء وتعديل الأخبار |
| `viewer` | تسجيل دخول + تصويت |
